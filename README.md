## Tasks Overview

### Task 1 — HTML Structure and Basic Server
- Built HTML forms for user input
- Set up Node.js server using Express
- Created server-side endpoints for form submissions
- Used EJS for server-side rendering

### Task 2 — Inline Styles, Validation and Server-Side Storage
- Extended forms with Name, Email, Age and Password fields
- Added client-side JavaScript validation with custom error messages
- Implemented server-side validation independently of browser
- Stored validated data in temporary server-side memory

### Task 3 — Advanced CSS Styling and Responsive Design
- Integrated Bootstrap 5 for responsive layout
- Added CSS animations and transitions
- Created gradient backgrounds, styled cards and hover effects
- Fully responsive across all screen sizes

### Task 4 — Complex Form Validation and Dynamic DOM Manipulation
- Built a 3-step multi-page form with dot navigation
- Added live password strength indicator (Very Weak to Very Strong)
- Character counter for name field
- Show/hide password toggle
- Client-side routing between form steps without page reload

### Task 5 — API Integration and Front-End Interaction
- Built a complete RESTful API with full CRUD operations
- GET /api/tasks — fetch all tasks
- POST /api/tasks — create a new task
- PUT /api/tasks/:id — update or complete a task
- DELETE /api/tasks/:id — delete a task
- Front-end Task Manager app built with vanilla JavaScript fetch API
- Features: add tasks, mark complete, edit, delete, filter by status/priority, live stats

- **Version Control:** Git and GitHub

## How to Run Any Task
1. Navigate into the task folder e.g. cd Task-5
2. Install dependencies: npm install
3. Start the server: node index.js
4. Open browser at: http://localhost:3000
